# 📌 Project Overview

**WeatherSense** is an interactive weather analytics and forecasting dashboard built using **Python**, **Streamlit**, and **Machine Learning**.

The application enables users to:

- 🌡 Analyze historical weather data
- 📊 Perform Exploratory Data Analysis (EDA)
- 🤖 Forecast future temperatures
- 📈 Visualize weather trends interactively
- ⚡ Generate insights using dynamic filters

---

# 🎯 Problem Statement

Weather datasets contain large volumes of climate information that can be difficult to analyze manually.

This project helps users:
- Understand temperature patterns
- Analyze precipitation trends
- Explore weather correlations
- Predict future temperature values

using interactive analytics and predictive modeling.
